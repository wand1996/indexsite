 Desenvolvimento de Software




Além de oferecer serviço de suporte e infraestrutura para nossos clientes, a GoFuture desenvolve software de Gestão ERP, tendo como principais clientes empresas do ramo do varejo e atacadista. O principal produto da nossa empresa hoje é o 'EasyDeal', um sistema robusto de controle de estoque para hipermercados, supermercados e mercados de médio porte.



Easy Deal :


O EasyDeal é um sistema ERP de gestão e controle de estoque, sendo este um sistema robusto programado em linguagem C Sharp, tendo compatibilidade para
os sistemas windows e Mac OS. Este sistema tem como funcionalidade guardar informações de produtos, cadastro, alteração e exclusão de produtos, volume de vendas, além de informações de colaboradores e loja. Nosso sistema é desenvolvido em duas versões: EasyDeal Bigger e Compact,
sendo a primeira para grandes centros comerciais e a compact, uma versão mais leve para pequenos e médios comerciantes.
